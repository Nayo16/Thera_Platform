# Therapy Platform 

## Description
Therapy Platform is an online mental health platform designed to improve access to therapy services in Ethiopia. Clients can book sessions with qualified therapists via chat, video call, or in-person meetings. The system also aims to reduce stigma around seeking mental health support and increase awareness about therapy.

---

## Features
- **User Authentication & Roles:** Clients, Therapists, Admins  
- **Session Booking:** Schedule chat, video, or in-person sessions  
- **Therapist Profiles:** Specialization, experience, and bio  
- **Role-Based Access Control:** Only authorized users can perform certain actions  
- **Admin Panel:** Manage users, monitor sessions, and approve therapists  
- **Secure Access:** JWT-based authentication  

---

## Tech Stack
- **Node.js** – JavaScript runtime for server-side development  
- **Express.js** – Backend framework for REST API and routing  
- **MongoDB** – NoSQL database (hosted on MongoDB Atlas)  
- **Mongoose** – ODM for MongoDB, handles schemas and database queries  
- **JWT (JSON Web Tokens)** – Authentication and secure route access  
- **Nodemon** – Automatically restarts server during development  
- **Postman** – Recommended for testing API endpoints
